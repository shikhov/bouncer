# Описание
Приложение для платформы Google App Engine, реализующее антиспам-бота для Telegram.

Механизм работы: если первое текстовое сообщение от нового пользователя (который впервые вошел в чат) похоже на спам (содержит ссылки либо определенные ключевые слова) — удалить сообщение и забанить пользователя. Если первое сообщение было без текста (фото, стикер и т.д.) — пользователь не считается спамером, но и легитимным тоже (до первого сообщения с текстом). Сообщение при этом не пересылается на forwards. Пользователи, уже находящиеся в чате на момент включения в него бота, считаются легитимными. Также бот удаляет сообщения о входе новых пользователей в чат.


# Настройка
Переименуйте прилагаемый example.config.py в config.py и отредактируйте его.

`TGTOKEN` — токен бота в Telegram

`TGMYCHATID` — id чата в Telegram, куда будут приходить уведомления о спаме, новых участниках чата и ошибках

`forwards` — dict, в котором ключ — это id чата в Telegram, а значение — URL, на который будут пересылаться все сообщения из чата, прошедшие проверку на спам (например, в приложение https://github.com/shikhov/vk2tg). Опциональный параметр.

# Развертывание
* Создать бота в Telegram, включить бота в чат, сделать администратором чата
* Отредактировать config.py
* Создать приложение в Google App Engine, задеплоить код
* Настроить вебхук телеграм-бота на адрес https://your_application.appspot.com/
* После настройки вебхука можно смотреть логи Google App Engine, там будут все приходящие json'ы в удобочитаемом виде, в них можно найти id чатов, пользователей и т.д.
